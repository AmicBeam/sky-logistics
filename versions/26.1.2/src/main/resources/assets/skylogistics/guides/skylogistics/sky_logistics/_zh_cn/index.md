# 天穹物流手册

天穹物流不是一格一格铺管道，而是通过命名的无线线路搬运资源。把节点放在箱子、储罐、电池或受支持的外部网络旁边，再用配置器让不同面的线路设置对应起来。

先阅读[祭坛与供奉](offerings.md)解锁核心材料，再搭建[物流节点](logistics.md)，最后用[工具与升级](tools.md)细调线路。

## 快速搭建

1. 制作或获得 <ItemLink id="eulogia_crystal" />，并通过供奉制作 <ItemLink id="chora_nectar" />。
2. 制作一个 <ItemLink id="configurator" /> 和至少两个 <ItemLink id="sky_node" />。
3. 一个节点贴着来源方块，另一个节点贴着目标方块。
4. 打开节点界面或使用配置器，把来源面设为抽取，把目标面设为插入，并设置到同一条线路。
