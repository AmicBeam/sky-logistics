# 节点与管道端点

<ItemLink id="sky_node" /> 是无线管道的端点。普通放置时默认存入，潜行放置时默认抽取。节点会贴在你点击的那一侧，只从那个方向访问相邻方块。

<RecipeFor id="sky_node" fallbackText="未找到天穹节点配方。" />

一次合成会得到 2 个节点。节点本身不存东西，只负责把同一条线路上的端点配对传输。

打开节点界面后，可以分别设置每一面：不连接、抽取或存入。只有连接好的面会参与线路。方块模型显示的是主要视觉模式，实际行为以每一面的设置为准。

节点还能选择搬运物品、流体或能量，设置目标优先级，并用红石控制开关。优先级高的存入端会优先接收；被红石条件暂停的面不会被当成失败端点。
